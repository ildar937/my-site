# NoirCard Studio — статический сайт (v2)

Готовая папка для GitHub Pages. Содержимое:

- `index.html` — лендинг (без зависимостей, чистый HTML+CSS).
- `assets/` — картинки-заглушки, favicon и `price.pdf` (замените своими).
- `.nojekyll` — отключает обработку Jekyll.

## Как развернуть на GitHub Pages

1. Создайте новый репозиторий (например, `noircard-site-v2`) ИЛИ используйте существующий.
2. Загрузите *весь* контент этой папки в корень репозитория.
3. В Settings → Pages выставьте: **Branch: main**, **Folder: /(root)**.
4. Откройте сайт: `https://<ваш_логин>.github.io/<имя_репозитория>/`.

### Что обязательно заменить
- В `index.html` вставьте ссылку на реальную **Google-форму** (кнопки «Оставить заявку», «Заказать»).
- В шапке и контактах замените WhatsApp и Telegram на свои ссылки.
- Замените содержимое `assets/` на свои изображения и `price.pdf`.
